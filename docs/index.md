# 欢迎使用 Togother Docs

这是您项目的文档站点，您可以在这里找到所有相关信息。

## 常用命令

* `mkdocs serve` - 启动实时重载的文档服务器
* `mkdocs build` - 构建文档站点
* `mkdocs -h` - 打印帮助信息并退出

## 项目结构

    mkdocs.yml    # 配置文件
    docs/
        index.md  # 文档首页
        ...       # 其他 Markdown 页面、图片和其他文件

## 功能特点

- 简洁易用的界面
- 强大的搜索功能
- 响应式设计，适配各种设备
- 支持深色模式
- 代码高亮显示
